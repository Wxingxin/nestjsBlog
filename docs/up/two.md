# 一、这个界面是什么？（你没理解错）

你现在看到的是 **VirtualBox 的「无人值守安装（Unattended Installation）」** 功能。

它的作用是：

* 自动创建用户
* 自动设置密码
* 自动装系统
* 自动装 Guest Additions
* 你几乎不用点 Ubuntu 安装界面

👉 **适合“我只想要一台能用的 Linux”**
👉 **不适合“我要学部署 / 学服务器 / 学运维”**

---

# 二、为什么我明确不建议你用 Unattended？

从「真实服务器学习」角度，问题非常大。

## ❌ 问题 1：用户、权限被 VirtualBox 偷偷改了

* 默认用户：`vboxuser`
* sudo 权限你不知道怎么来的
* 和云服务器创建用户流程 **不一致**

以后你会问：

> 为什么我在阿里云上不是这样？

---

## ❌ 问题 2：SSH / 网络 / 安装过程你完全没经历

你现在学的是：

* Linux
* Docker
* Nginx
* 服务器部署

👉 **而不是“一键生成 Linux”**

Unattended 会让你：

* 不知道 SSH 是什么时候装的
* 不知道网络是怎么配的
* 不知道系统盘是怎么分的

---

## ❌ 问题 3：Guest Additions 对 Server 来说是“干扰项”

对于 **Ubuntu Server（无界面）**：

* 不需要共享剪贴板
* 不需要自动分辨率
* 不需要窗口缩放

👉 **Guest Additions 对服务器学习毫无必要**
