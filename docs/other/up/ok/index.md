# 第一步：查看虚拟机 IP 地址（现在做）

在你现在看到的这个终端里，直接执行：

```bash
ip a
```

你会看到一堆网络信息，重点找 **不是 lo 的那一段**，例如：

```
inet 192.168.x.x/24
```

或（如果是 NAT）：

```
inet 10.0.2.15/24
```

👉 把这个 **IP 地址记下来**（或者直接复制发给我）。

---

# 第二步：用你自己的电脑 SSH 连这台服务器（核心）

等你拿到 IP 后，在 **你自己的电脑（不是虚拟机）**上操作：

### Windows（PowerShell / CMD 都行）或 macOS / Linux 终端：

```bash
ssh wjx@你的虚拟机IP
```

例如：

```bash
ssh wjx@192.168.1.120
```

### 第一次连接会提示：

```
Are you sure you want to continue connecting (yes/no)?
```

输入：

```
yes
```

然后输入你刚才的登录密码。

👉 如果这一步成功，说明：

* SSH 正常
* 网络正常
* 你可以 **完全不用再看 VirtualBox 的黑窗口**

---

# 第三步（马上要做）：更新系统（非常重要）

无论你是否已经 SSH 进来，**第一件事永远是更新系统**：

```bash
sudo apt update
sudo apt upgrade -y
```

输入密码即可。
