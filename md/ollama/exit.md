在 **Ollama 的 `ollama run qwen...` 终端对话框**里，退出方式取决于你现在所处的状态。下面按**可靠程度从高到低**给你说明。

---

## 一、最标准、最通用的退出方式（推荐）

### **`Ctrl + D`（EOF，首选）**

* 适用：**macOS / Linux / Windows（WSL）**
* 含义：发送 EOF（文件结束）
* 效果：**立即退出当前模型对话，回到 Shell**

> 这是 Ollama 官方交互里**最稳定**的退出方式。

---

## 二、强制中断（如果模型正在输出）

### **`Ctrl + C`**

* 第一次按：**中断当前生成**
* 连续按 1–2 次：**直接退出对话**

如果你看到模型还在“疯狂输出”，直接 `Ctrl + C`。

---

## 三、输入命令方式（不一定每个版本都支持）

在输入框里直接输入：

```
/exit
```

或

```
/quit
```

⚠️ 注意：
并非所有 Ollama 版本/模型都支持 slash 命令，**如果没反应，属正常情况**。

---

## 四、最“暴力”的方式（兜底）

### 关闭终端窗口

* 100% 有效
* 不推荐作为日常方式

---

## 五、常见误区

* ❌ 输入 `exit`（没用）
* ❌ 输入 `quit()`（这是代码，不是命令）
* ❌ 以为是 Vim（不是）

---

## 六、总结（你只记这一行就够）

> **退出 Ollama 模型对话：优先用 `Ctrl + D`，不行就 `Ctrl + C`**
